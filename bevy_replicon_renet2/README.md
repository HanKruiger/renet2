# Bevy Replicon Renet2

An integration of `bevy_renet2` as a messaging backend for [`bevy_replicon`](https://github.com/projectharmonia/bevy_replicon).


## Compatible versions

| bevy_renet2 | bevy_replicon | bevy_replicon_renet2 |
| ----------- | ------------- | -------------------- |
| 0.0.3       | 0.25          | 0.0.3                |
